# Media Library

This folder contains static assets for the Bridgewater Democratic Town Committee website.

## Files needed:

### DTC_Donor_Cert_Form.pdf
The Connecticut SEEC Individual Contributor Certification Form should be placed here. This PDF contains the legal form that donors must fill out before making contributions as required by Connecticut state law.

The form includes:
- Individual contributor information fields
- Donation amount
- Address information
- Employment details
- Legal disclosure questions about lobbying and state contracts
- Digital signature certification

This file should be accessible at `/media/DTC_Donor_Cert_Form.pdf` for download by users who prefer to mail in checks rather than donate online.
